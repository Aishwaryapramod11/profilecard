# Profile Card

An elegant, clean, and responsive digital profile card for **Aishwarya Pramod Nair**, customized with a profile photo, links, and interaction handlers.

## 🚀 Features
- **Clean Aesthetic**: Centered white card with rounded corners, drop shadows, and rose pink accents.
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.
- **Social Integration**: Connected to GitHub and LinkedIn profiles via clean, modern SVGs.
- **Micro-Interactions**: Features click toggle states for the "Follow" action.

## 🛠️ Tech Stack
- **HTML5**: Structured semantic markup.
- **CSS3**: Layout, spacing, border styles, custom fonts, and responsiveness.
- **JavaScript (ES6+)**: Lightweight button interaction handlers.

## 💻 Running Locally
To view the page locally, simply open `index.html` in any browser or start a local server:
```bash
# Python 3
python -m http.server 8080
```
